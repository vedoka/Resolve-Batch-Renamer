# Resolve-Batch-RenamerDaVinci Resolve Timeline Clip Renamer

A small utility script for DaVinci Resolve that automatically renames timeline clips, eliminating the need to rename them one by one.
Especially useful for speeding up repetitive tasks in VFX workflows.

Installation

Simply copy the script file to the following directory:

C:\Users\Vedoka_WST2\AppData\Roaming\Blackmagic Design\DaVinci Resolve\Support\Fusion\Scripts\Edit


After placing the script in this folder, it will appear in the Workspace → Scripts → Edit menu inside DaVinci Resolve.
